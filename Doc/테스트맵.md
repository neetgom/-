🌙 [Test Map] 원신 6.3 '달 결정' 기능 테스트 맵



1. ⚡️ 발동 조건 (Trigger Conditions)

1.1. 💧 물 + 🪨 바위 반응 (Water + Geo Reaction)

1.1.1 반응 즉시성: 물 원소와 바위 원소 접촉 시 반응이 즉시 발동하는지 확인

1.1.2 부착 순서 검증: 원소 부착 순서(물→바위 vs 바위→물)가 바뀌어도 동일하게 반응하는지 확인

1.2. 🚫 방해 요소 확인 (Interference Factors)

1.2.1 타 원소 개입: 불🔥, 얼음❄️, 번개⚡️ 등 타 원소 개입 시 반응 우선순위 및 오작동 여부

1.2.2 다중 반응 로직: 다중 원소 부착 상태에서의 반응 처리 로직 확인 (예: 증발 vs 달 결정)

2. ✨ 조각 생성 (Crystal Creation)

2.1. 💎 달빛 조각 생성 (Moonlight Fragment Generation)

2.1.1 생성 확인: 반응 성공 시 필드에 '달빛 조각' 1개가 정상적으로 생성되는지 확인

2.1.2 위치 유효성: 조각이 지형 아래나 공중 등 비정상 위치에 생성되지 않는지 확인

2.1.3 지속 시간: 생성된 조각이 10~15초 동안 유지되며, 시간 경과 후 정상 소멸하는지 확인

3. 🧲 조각 수집 (Crystal Collection)

3.1. 📥 획득 (Acquisition)

3.1.1 자동 획득: 캐릭터가 근처(약 5m) 접근 시 자동으로 획득되는지 확인

3.1.2 획득 주체: 현재 필드에 나와 있는 Active 캐릭터만 획득 가능한지 확인 (대기 캐릭터 불가)

3.2. 🔢 누적 (Accumulation)

3.2.1 최대 스택: 캐릭터 주위에 최대 3개까지 조각이 정상적으로 누적되는지 확인

3.2.2 초과 처리: 3개 초과 획득 시 기존 조각이 갱신되거나 획득 불가 처리되는지 확인

4. ⚔️ 자동 공격 (Auto Attack)

4.1. 💥 3스택 발동 (3-Stack Activation)

4.1.1 즉시 발동: 조각 3개 수집 즉시 자동 공격이 발동하는지 확인

4.1.2 이펙트 출력: 스택 소모와 함께 공격 이펙트가 정상 출력되는지 확인

4.2. 🎯 타겟팅 (Targeting)

4.2.1 유도 성능: 공격이 **'가장 가까운 적'**을 향해 정확히 유도되는지 확인

4.2.2 다수 타겟: 적이 다수일 경우 타겟팅 정확도 및 엉뚱한 방향 발사 여부 확인

4.2.3 예외 처리: 적이 없을 때(비전투)의 발동 처리 확인 (허공 발사 또는 대기 상태 유지)

5. 📉 데미지 (Damage)

5.1. 📊 수치 검증 (Value Verification)

5.1.1 원소 마스터리: 캐릭터의 원소 마스터리 수치 증가에 따른 데미지 상승 확인

5.1.2 치명타 적용: 치명타(Critical) 확률/피해 적용 여부 및 데미지 폰트 출력 확인

5.1.3 속성 내성: 적의 속성 및 바위 원소 내성에 따른 데미지 감소/증폭 계산 확인

6. 🧩 기타 (Others)

6.1. ⚖️ 속성 판정 (Attribute Judgment)

6.1.1 오브젝트 속성: 달빛 조각이 '바위 창조물' 속성으로 인식되는지 확인

6.1.2 스킬 상호작용: 타 캐릭터(예: 치오리 등)의 바위 창조물 연계 스킬과 정상 상호작용 여부 확인
